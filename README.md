# Estude PAS

Web app para estudar questões do PAS (Programa de Avaliação Seriada) da UnB.

## Funcionalidades

- 40 questões da prova PAS 1 de 2024
- Filtros por ano e matéria
- Questões tipo C (Certo/Errado) e tipo A (múltipla escolha)
- Feedback imediato ao responder
- Estatísticas de acertos/erros por matéria
- Dark mode
- Atalhos de teclado
- Progresso salvo no navegador

## Stack

- React 19
- Vite
- Tailwind CSS 4

## Rodando o projeto

```bash
npm install
npm run dev
```

Acesse http://localhost:5173

## Fonte das questões

Provas oficiais do CEBRASPE disponíveis em cdn.cebraspe.org.br
